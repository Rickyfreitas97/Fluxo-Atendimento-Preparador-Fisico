# Fluxo de Atendimento Automatizado para Preparador Físico 🤖💪

Este projeto consiste na arquitetura e desenvolvimento de um fluxo inteligente de atendimento automatizado (Chatbot) voltado para o WhatsApp, configurado especificamente para as necessidades de um **Preparador Físico / Personal Trainer**. 

O objetivo é otimizar a captação de alunos, automatizar a triagem inicial e agilizar o envio de informações sobre consultoria e treinos de forma escalável.

## 🧠 Engenharia de Prompt & Concepção
Toda a árvore de decisão, roteiros de conversação humanizados e gatilhos de mensagens foram inteiramente concebidos e refinados através de **Inteligência Artificial**. Utilizando técnicas avançadas de **Engenharia de Prompt**, as instruções foram estruturadas para garantir uma abordagem acolhedora, objetiva e altamente eficiente, eliminando gargalos no atendimento manual.

## 🚀 Principais Funcionalidades do Fluxo
* **Triagem de Pré-Avaliação (Anamnese Básica):** Coleta automatizada de dados essenciais do cliente (objetivo como emagrecimento ou hipertrofia, histórico de treinos).
* **Apresentação de Planos & Consultoria:** Menu interativo detalhando as modalidades (como Funcional, Musculação e Híbrido) com seus respectivos valores (Básico, Intermediário e Premium).
* **Direcionamento Estratégico:** Encaminhamento inteligente para o fechamento ou suporte direto no bloco final de contato ("Falar com Preparador").
* **Respostas Rápidas (FAQ):** Respostas automáticas sobre o funcionamento da consultoria e dinâmica dos treinos.

## 🛠️ Ferramentas e Conceitos
* **Plataforma de Construção:** FunilPRO
* **Concepção:** Inteligência Artificial & Engenharia de Prompt Estruturada
* **Nicho de Mercado:** Saúde, Fitness e Alta Performance
* **Área de Aplicação:** Automação de Processos (RPA) / Vendas e CRM
* **Versionamento:** Git & GitHub
